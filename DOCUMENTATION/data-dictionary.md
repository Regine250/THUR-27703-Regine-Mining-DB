# Data Dictionary.

This document describes all tables in the database, their columns, types, constraints and purpose.

## Conventions & Definitions  
- Data types and sizes reflect how data is stored in the database.  
- PK = Primary Key.  
- FK = Foreign Key.  
- Nullable = whether column allows NULL.  
- Default = Default value, if any.  
- Description = Purpose / meaning / business use.
